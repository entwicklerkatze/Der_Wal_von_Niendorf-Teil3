### 2. Video-Dauer Anomalie

**Externe Behauptung:**
> "Dateiname suggeriert: 20-Minuten-TV-Sendung (12:40-13:00)  
> Tatsächliche Dauer: 46.6 Sekunden  
> Erstellungszeitpunkt: 07:47 Uhr (Morgens, nicht zur 'Sendezeit')"

**Unsere Analyse:**
```
Dateiname: TV-20260326-1240-1300_Robert_Marc_Lehmann_1080.mp4
Impliziert: 20 Minuten (12:40 - 13:00)

Dateigröße: 25,797,836 Bytes (24.6 MB)
Berechnung: 24.6 MB ÷ ~0.8 MB/s (1080p) = ~30.75 Sekunden
```

**FAZIT:** 🟡 **TEILWEISE BESTÄTIGT**
Die Dateigröße deutet auf eine tatsächliche Dauer von ~31 Sekunden hin, nicht 20 Minuten. Dies ist eine signifikante Diskrepanz.
